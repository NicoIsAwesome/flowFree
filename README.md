## Usage
work with python2

### [1] edit "game" in flowFree.py
![alt text](https://github.com/NicoIsAwesome/flowFree/blob/main/flowFree_1.jpg)
```
game = [1, 0, 0, 0, 2, \
        0, 0, 0, 0, 0, \
        0, 3, 4, 0, 0, \
        0, 0, 1, 0, 0, \
        4, 3, 2, 0, 0 ]
Note:
 0 = space
```
